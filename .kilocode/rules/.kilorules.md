## 🛡️ Security Laws (Zero Leak Policy)
1. **PROTECTION:** NEVER read the `.env` file directly. Use `.env.example` to understand variable structures.
2. **ISOLATION:** Keep all secrets (API keys, bank tokens) on the host machine only.
3. **LOGS:** Never output raw transaction data or keys in terminal logs or UI.

3. **GRID:** Use strict alignment. Data shouldn't "jump" when updated.
